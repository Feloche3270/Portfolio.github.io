Ce projet représente la suite du stage après avoir effectué l'audit de l'intranet.
Cette mission quant à elle avait pour but de créer un module organigramme intéractif via la recherche d'agent et permettant de visualiser facilement quel agent était relié à un service/agent grâce à un nom de servies, de la personne, une photo et une icône informations permettant de connaitre le téléphone, l'adresse, la localisation, rappel du service et le nombre d'agent sous sa responsabilité.



Avant de commencer le projet j'ai conçu une maquette via Figma afin de me donner une idée de ce à quoi devait ressembler l'organigramme.
Cela m'a permis d'avoir une idée claire de où je voulais aller !
Voici à quoi ressemblait le figma une fois celui-ci fini.


Pour réaliser cet organigramme il fallait des compétences en JS que je n'avais pas, je me suis demandé si j'avais le temps d'apprendre cette technologie mais étant bien trop vaste j'ai décidé de chercher un module organigramme déjà en ligne.
Les recherches ont donc commencée afin de trouver un module organigramme facile à prendre en main et facilement modifiable.

Le module organigramme libre de droit de Dabeng m'a permis d'effectuer l'organigramme dans les temps.
Il a fallut installer les dépendances de Node.js afin de pouvoir utiliser et créer l'organigramme.

Voici un extrait de code permettant de comprendre la structure de code de l'organigramme et le rendu final une fois sur votre page internet.


L'organigramme est réductible ou dépliable suivant votre choix et une fonction recherche en haut de page est disponible pour trouver facilement l'agent ou le service souhaité.





Ce projet m'a premièrement permis de mettre les mains dans du code js via la dépendance Node.js
C'est une technologie que nous ne voyons pas en cours et cela m'a permis de découvrir un nouveau langage et d'augmenter mes compétences.
Il a aussi fallut que je me renseigne sur ce langage et sur le code d'un autre développeur en comprenant les subtilités de celui-ci et ce fût une belle expérience.
